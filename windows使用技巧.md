### 快捷键

首先要理解快捷键的逻辑：为什么windows键盘上会有`win`、`Alt`、`Ctrl`和`Shift`键？

+ `win + ;`——打开表情输入🤞
+ `win + Q/S`——打开系统搜索框
+ `win + E`——打开文件资源管理器
+ `win + R`——打开运行
+ `win + U`——进到系统设置并定位在搜索框
+ `win + I`——进到设置并定位在搜索框
+ `win + P`——打开投影设置
+ `win + A`——打开通知
+ `win + D`——显示桌面
+ `win + G`——打开录屏
+ `win + K`——搜索连接设备
+ `win + L`——锁屏
+ `win + X`——相当于右击windows图标
+ `win + V`——打开剪贴板
+ `win + <`——显示桌面，`win`键松开返回当前应用
